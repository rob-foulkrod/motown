# 🎵 Motown 🎶

> *Get the band back together!* Motown is a snappy little **ASP.NET Core MVC** web app, ready to take the stage. 🎤

## 🌟 Overview

**Motown** is a web application built on the rock-solid **ASP.NET Core MVC** framework. It started life as a hands-on project for learning Git and modern .NET web development, and it's grown into a clean, ready-to-extend foundation you can riff on. Whether you're here to learn, hack, or jam on new features, Motown gives you a tidy stage to perform on. 🎸

## 🎚️ Prerequisites

Before you start the show, make sure you've got the gear:

- 🧰 **[.NET SDK 10.0](https://dotnet.microsoft.com/download/dotnet/10.0)** or newer
- 🌐 A modern web browser (to enjoy the encore)

Check that the .NET SDK is installed and on the right track:

```bash
dotnet --version
```

## 🚀 Build & Run

Time to plug in and play! All commands run from the **`motown.web`** project folder.

1. **Clone the repo** and step into the project:

   ```bash
   git clone https://github.com/rob-foulkrod/motown.git
   cd motown/motown.web
   ```

2. **Restore the dependencies** (warm up the band):

   ```bash
   dotnet restore
   ```

3. **Build the project** (sound check):

   ```bash
   dotnet build
   ```

4. **Run the app** (showtime! 🎉):

   ```bash
   dotnet run
   ```

5. 🌍 Open your browser and head to **<http://localhost:5000>** (or the HTTPS URL printed in your console) to see Motown live in concert.

To stop the show, just press `Ctrl+C` in the terminal. 👋
